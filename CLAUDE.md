# Проект: Лендинг «Живая форма»

Лендинг-портфолио для мастера бетонных кашпо на заказ. Один мастер, ручная работа, Москва. Аудитория — люди, которые думают об интерьере как о продолжении себя.

Слоган: **«Ручная работа. Живая форма.»**

---

## Стек

- HTML / CSS / Vanilla JS (без фреймворков)
- Vite (сборщик)
- Lenis (smooth scroll)
- GSAP + ScrollTrigger + SplitText (анимации)
- Google Fonts: Fraunces (variable serif, заголовки) + DM Sans (тело)

---

## Дизайн-система

**Палитра — Warm Brutalist:**
```
--color-base:    #E8E2D9   /* тёплый гриж, основной фон */
--color-surface: #B5AFA6   /* средний тёплый серый */
--color-accent:  #C4694F   /* терракота — единственный яркий цвет */
--color-deep:    #2E2925   /* угольно-коричневый */
--color-whisper: #F5F0EA   /* off-white для текста на тёмном */
```

**Типографика:**
- Заголовки: Fraunces, hero ~120px (`clamp(64px, 12vw, 160px)`)
- Тело: DM Sans

**Правило акцента:** терракота `#C4694F` — только CTA-кнопки и единичные акценты. Нигде больше.

---

## Архитектура страницы (9 блоков)

1. **Hero** — полный экран, lifestyle-фото, parallax, CTA скроллит к галерее
2. **Галерея** — horizontal scroll pin (GSAP), 8+ изделий, каждое 100vw
3. **Доверие №1** — 4 цифры (200+, 4.9/5, 3–4 нед., с 2019) + одна цитата
4. **О мастере** — фото рук за работой, текст от первого лица
5. **Материалы** — 4 карточки: состав, цвет, дренаж, защита
6. **Как заказать** — 4 шага, тёмный фон
7. **Отзывы** — 3 карточки с UGC-фото из реальных домов
8. **Стоимость** — 3 ценовых диапазона (от 2 500 / 5 500 / 10 000 ₽)
9. **Финальный CTA** — тёмный фон, кнопка Telegram, sticky на мобайле

---

## Анимации (в порядке приоритета)

1. Text reveal на заголовках (SplitText, stagger 120ms, clip-mask снизу)
2. Clip-path image reveal (600ms, expo.out)
3. Horizontal gallery pin
4. Magnetic hover на CTA (QuickTo, радиус 80px)
5. Custom cursor (30px → 80px «смотреть», только десктоп)
6. Counter animation на цифрах блока доверия

**Easing везде:** `power2.out` или `expo.out`. Никакого `bounce`.
