## 1. ЦЕЛЬ ПРОЕКТА

### Бизнес-цель
Генерировать входящие заявки на изготовление кастомных бетонных кашпо. Единственное конверсионное действие на всей странице — **начать диалог в Telegram или WhatsApp**.

### Задачи сайта
1. Убедить посетителя в качестве изделий через визуал и портфолио
2. Создать доверие к мастеру как к конкретному человеку
3. Снять 6 главных возражений (качество, хрупкость, цена, срок, подходимость, надёжность)
4. Упростить первый шаг до одного нажатия кнопки

### Конверсионное действие
```
Нажатие на "Написать в Telegram" или "Написать в WhatsApp"
```
Никаких промежуточных шагов. Никаких длинных форм. Никаких email-рассылок.

### Вторичная цель
Для тех, кто не готов писать сразу — подписка на Instagram. Это длинная воронка.

### KPI запуска
- Конверсия в клик на Telegram/WhatsApp: **≥ 3%** от уникальных посетителей
- Время на сайте: **≥ 90 секунд** (значит, читают, а не уходят)
- Bounce rate мобильных: **≤ 55%**

## 2. ЦЕЛЕВАЯ АУДИТОРИЯ

### Основной сегмент: B2C — частные покупатели

| Параметр | Описание |
|----------|----------|
| Возраст | 25–45 лет |
| Гео | Крупные города России, прежде всего Москва и Санкт-Петербург |
| Интересы | Интерьер, растения, дизайн, japandi / wabi-sabi / скандинавская эстетика |
| Мотивация | Хочет объект, который живёт в доме десятилетиями. Не очередной масс-маркет |
| Боль | Все горшки одинаковые. Хочу что-то под конкретное место и растение |
| Канал прихода | Instagram, рекомендации, поиск |

### Вторичный сегмент: B2B — профессионалы
- Интерьерные дизайнеры (нестандартные размеры, серийные заказы)
- Кафе, отели, рестораны, офисы
- Флористы и магазины декора

### Психографика принятия решения
Покупатель платит выше рынка за хендмейд-объект, которого не видел и не трогал. Главный страх — разочарование. Главный триггер — уверенность в том, что мастер **понял**, что нужно, и **умеет** это сделать.

## 3. КОНЦЕПЦИЯ И ПОЗИЦИОНИРОВАНИЕ

### Центральная идея
> **Бетон — самый постоянный материал. Растение — самое живое.**
> Кашпо — единственное место, где они существуют вместе.

Это противоречие (тяжёлое/живое, серое/зелёное, неизменное/растущее) — сквозная метафора сайта. Она определяет выбор цветов, скорость анимаций, тон текста и стиль фотографии.

### Слоган (используется как H1 на странице)
```
Ручная работа. Живая форма.
```

### Позиционирование бренда
- **Кто:** Один мастер. Не студия, не цех, не агрегатор
- **Для кого:** Люди, которые думают об интерьере как о продолжении себя
- **Против чего:** Анонимного производства и "цены по запросу" без объяснений
- **Голос:** Первое лицо, конкретика, без корпоративного языка

### Tone of Voice — правило конкретики

| ❌ Не писать | ✅ Писать |
|-------------|----------|
| Мы стремимся к качеству | Я выбраковываю изделие с любым пузырём |
| Высококачественные материалы | Бетон М400 с фиброволокном — не трескается |
| Индивидуальный подход | Уточню, что у вас растёт и где стоит горшок |
| Быстрая доставка | 3–4 недели. Фото до отправки |
| Отличное качество, рекомендую! | Ждала дольше, чем планировала, но результат — ровно то, о чём мы говорили |

## 4. ДИЗАЙН-СИСТЕМА

### 4.1 Цветовая палитра

Направление: **Warm Brutalist** — тёплый, а не холодный. Бетон = крафт, не стройка.

```css
:root {
  --color-base:    #E8E2D9; /* тёплый гриж, бетонная пыль — основной фон        */
  --color-surface: #B5AFA6; /* средний тёплый серый, сырой бетон                 */
  --color-accent:  #C4694F; /* терракота, обожжённая глина — ЕДИНСТВЕННЫЙ яркий  */
  --color-deep:    #2E2925; /* угольно-коричневый (тёмные секции, не чистый #000) */
  --color-whisper: #F5F0EA; /* лён / off-white — текст на тёмном фоне            */
}
```

**Правило акцента:** `--color-accent` используется только для кнопок-CTA и числовых акцентов в статистике. Нигде больше. Один акцентный цвет — максимум внимания в нужных точках.

**Текстура фона:** Тонкий grain-оверлей (SVG-фильтр `feTurbulence` или PNG-текстура с `opacity: 0.04`) имитирует шероховатость бетона на светлых секциях. CSS:
```css
.section--light::before {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url('/assets/grain.png');
  background-size: 200px;
  opacity: 0.04;
  pointer-events: none;
}
```

### 4.2 Типографика

```css
:root {
  --font-display: 'Fraunces', serif;    /* заголовки                     */
  --font-body:    'DM Sans', sans-serif; /* весь остальной текст           */
}
```

**Fraunces** — variable serif с осью `wdth` и `wght`. Выглядит рукотворным, "странным" — именно это нужно для артизан-бренда. Загрузка с Google Fonts:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Fraunces:ital,opsz,wght@0,9..144,300..900;1,9..144,300..900&family=DM+Sans:wght@300;400;500&display=swap" rel="stylesheet">
```

**Масштаб шрифтов:**

| Элемент | Размер | Вес | Шрифт |
|---------|--------|-----|-------|
| Hero H1 | `clamp(56px, 10vw, 140px)` | 600 | Fraunces |
| Section H2 | `clamp(36px, 6vw, 80px)` | 500 | Fraunces |
| H3 карточки | `20–24px` | 500 | Fraunces |
| Body large | `18–20px` | 300 | DM Sans |
| Body base | `15–16px` | 400 | DM Sans |
| Caption | `12–13px` | 400 | DM Sans |
| Цифры-статистика | `clamp(40px, 6vw, 72px)` | 600 | Fraunces |

**Правило контраста:** В ключевых местах текст либо очень большой (hero, заголовки секций), либо очень маленький (подписи, детали). Никаких "средних" размеров в акцентных позициях.

### 4.3 Сетка и отступы

```css
:root {
  --grid-max:     1280px;
  --grid-padding: clamp(20px, 5vw, 80px); /* боковые отступы */
  --section-gap:  clamp(80px, 12vw, 160px); /* между секциями */
}
```

Секции чередуются: светлый фон (`--color-base`) → тёмный фон (`--color-deep`). Это создаёт ритм и делает страницу нескучной при скролле.

### 4.4 Стиль фотографии

**Главное правило:** Живые растения в реальных домах. Никогда — пустое кашпо на белом фоне студии.

**Четыре типа кадров (все обязательны):**

| Тип | Описание | Применение |
|-----|----------|-----------|
| **Lifestyle** | Кашпо в жилом пространстве, оконный свет, ≥70% кадра — контекст | Hero, галерея |
| **Texture macro** | Поверхность бетона так крупно, что становится абстракцией. Видна каждая пора | Блок материалов, фон секций |
| **Process** | Руки в бетоне, инструмент, форма. НЕ постановочное "улыбающееся лицо" | Блок о мастере |
| **Scale** | Кашпо рядом с рукой / монетой / известным предметом | Подписи в галерее |

**Обработка:** Одинаковая для всех фото. Тёплый тон, не перенасыщенный. Лёгкая плёночная зернистость — соответствует текстуре бетона.

### СЕКЦИЯ 01: HERO

**Задача:** Удержать внимание за первые 3 секунды. Сформулировать УТП. Спровоцировать скролл.

**Фон:** Полноэкранное lifestyle-фото. Кашпо с растением, подоконник или полка, естественный свет. Тонкий тёмный gradient overlay снизу для читаемости текста.

**HTML-структура:**
```
section.hero
  ├── div.hero__bg          ← фото + grain overlay
  ├── div.hero__content
  │     ├── h1.hero__title  ← "Ручная работа. / Живая форма."
  │     ├── p.hero__sub     ← подзаголовок
  │     └── a.btn--primary  ← "Смотреть работы ↓"
  └── div.hero__scroll-hint ← анимированная стрелка вниз
```

**Текст:**

```
H1 (Fraunces, ~120px, weight 600, --color-whisper):
Ручная работа.
Живая форма.

P (DM Sans, 18px, weight 300, --color-whisper с opacity 0.75):
Бетонные кашпо на заказ — каждое под конкретный интерьер.
Делаем в Москве. Отправляем по всей России.

КНОПКА (терракота, --color-accent):
Смотреть работы ↓
```

**Поведение кнопки:** Anchor-ссылка `#gallery`. Плавный скролл через `lenis.scrollTo('#gallery')`. Не открывает форму — холодный посетитель ещё не готов к заявке.

**Параллакс:** Фоновое изображение движется на `0.3x` скорости скролла. Текст движется на `1x`. Реализация: GSAP ScrollTrigger `scrub: true`, `yPercent: -20` на `.hero__bg`.

### СЕКЦИЯ 02: ГАЛЕРЕЯ

**Задача:** Показать диапазон и качество. Превратить любопытство в желание. Это главная "продающая" секция.

**Компоновка:**
- **Десктоп:** Pinned horizontal scroll. Секция занимает `100vh`, внутри скролится горизонтальный трек. Каждый кадр = `100vw`. GSAP ScrollTrigger с `pin: true`.

**Минимум 8 изделий.** Обязательное разнообразие: разные формы (цилиндр, параллелепипед, конус, неправильная), разные размеры, 3–4 цветовых варианта.

### СЕКЦИЯ 04: О МАСТЕРЕ

**Задача:** Превратить бренд в человека. При покупке хендмейда покупают мастера.

**Фон:** `--color-base` (светлый).

**Компоновка:** 2 столбца, 50/50.

**Слева:** Фото. Руки в бетоне, форма или инструмент. НЕ портрет в духе LinkedIn. НЕ улыбающийся человек в фартуке на белом фоне.

**Справа:** Текст.

```
H2 (Fraunces):
[Имя мастера]

BODY (DM Sans, 16px, line-height 1.7):
Меня зовут [имя]. Я работаю с бетоном с [год].

Сначала это было хобби — мне нравилось, что из серой
массы можно вытащить точную, чистую форму. Потом начал
делать кашпо для себя. Потом для друзей. Потом понял,
что это то, чем хочу заниматься.

Я выбраковываю любое изделие с пузырями или трещинами,
даже если это стоит лишнего дня работы. Не потому что
так положено — а потому что иначе не могу.

Каждое кашпо — отдельный проект. Уточняю, куда оно
пойдёт, что в нём будет расти, какое освещение.
Это важно: форма и цвет должны работать именно там.
```

**4 факта под текстом (горизонтально, DM Sans 13px, caps):**
```
РУЧНАЯ РАБОТА        МИНЕРАЛЬНЫЕ        ФОТО ДО          ДОСТАВКА
НА КАЖДОМ ЭТАПЕ      ПИГМЕНТЫ           ОТПРАВКИ         ПО ВСЕЙ РОССИИ
```

## 7. АНИМАЦИИ И ИНТЕРАКТИВНОСТЬ

### 7.1 Технический стек анимаций

```javascript
// Обязательные зависимости
import Lenis from '@studio-freight/lenis';          // smooth scroll
import { gsap } from 'gsap';                         // анимации
import { ScrollTrigger } from 'gsap/ScrollTrigger'; // scroll-linked
import { SplitText } from 'gsap/SplitText';         // text reveals (free 3.12+)
```

**Базовая инициализация (всегда первой):**
```javascript
const lenis = new Lenis({ duration: 1.2, easing: t => Math.min(1, 1.001 - Math.pow(2, -10 * t)) });
lenis.on('scroll', ScrollTrigger.update);
gsap.ticker.add(time => lenis.raf(time * 1000));
gsap.ticker.lagSmoothing(0);
```

**Easing по умолчанию:** `power2.out` для большинства анимаций. `expo.out` для reveal-эффектов. **Никогда:** `bounce`, `elastic`.

---

### 7.2 Анимации по приоритету

#### ПРИОРИТЕТ 1 — Text Reveal на заголовках

Применяется к: H1 в герое + H2 в секциях 03, 04, 09.

```javascript
// Для каждого .reveal-title
const split = new SplitText('.reveal-title', { type: 'lines' });
split.lines.forEach(line => {
  const wrapper = document.createElement('div');
  wrapper.style.overflow = 'hidden';
  line.parentNode.insertBefore(wrapper, line);
  wrapper.appendChild(line);
});

gsap.from(split.lines, {
  y: '110%',
  opacity: 0,
  duration: 0.9,
  ease: 'expo.out',
  stagger: 0.08,
  scrollTrigger: { trigger: '.reveal-title', start: 'top 85%' }
});
```

#### ПРИОРИТЕТ 2 — Clip-path Image Reveal

Применяется к: всем изображениям кашпо в секции материалов, блоке о мастере, отзывах.

```javascript
gsap.from('.reveal-image', {
  clipPath: 'inset(0 0 100% 0)',
  duration: 1.0,
  ease: 'expo.out',
  stagger: 0.12,
  scrollTrigger: { trigger: '.reveal-image', start: 'top 80%' }
});
```

CSS для изображений:
```css
.reveal-image { clip-path: inset(0 0 0% 0); will-change: clip-path; }
```

#### ПРИОРИТЕТ 3 — Horizontal Gallery Pin

```javascript
const galleryTrack = document.querySelector('.gallery__track');
const cards = gsap.utils.toArray('.gallery__card');

gsap.to(galleryTrack, {
  xPercent: -100 * (cards.length - 1),
  ease: 'none',
  scrollTrigger: {
    trigger: '.gallery',
    pin: true,
    scrub: 1,
    snap: 1 / (cards.length - 1),
    end: () => '+=' + galleryTrack.offsetWidth
  }
});
```

Прогресс-индикатор:
```javascript

  scaleX: 1,
  ease: 'none',
  scrollTrigger: { trigger: '.gallery', scrub: true, pin: true }
});
```

#### ПРИОРИТЕТ 4 — Counter Animation (секция доверия)

```javascript
document.querySelectorAll('.stat-number').forEach(el => {
  const target = +el.dataset.target; // data-target="200"
  const obj = { val: 0 };
  gsap.to(obj, {
    val: target,
    duration: 1.5,
    ease: 'power2.out',
    onUpdate: () => { el.textContent = Math.round(obj.val) + (el.dataset.suffix || ''); },
    scrollTrigger: { trigger: el, start: 'top 85%', once: true }
  });
});
```

HTML:
```html
<span class="stat-number" data-target="200" data-suffix="+">0</span>
```

#### ПРИОРИТЕТ 5 — Magnetic Hover на кнопках

Применяется к: главным CTA-кнопкам `.btn--primary`. Только десктоп.

```javascript
// Только если есть hover (десктоп)
if (window.matchMedia('(hover: hover) and (pointer: fine)').matches) {
  document.querySelectorAll('.btn--magnetic').forEach(btn => {
    const xTo = gsap.quickTo(btn, 'x', { duration: 0.4, ease: 'power2.out' });
    const yTo = gsap.quickTo(btn, 'y', { duration: 0.4, ease: 'power2.out' });

    btn.addEventListener('mousemove', e => {
      const rect = btn.getBoundingClientRect();
      const cx = rect.left + rect.width / 2;
      const cy = rect.top + rect.height / 2;
      xTo((e.clientX - cx) * 0.35);
      yTo((e.clientY - cy) * 0.35);
    });
    btn.addEventListener('mouseleave', () => { xTo(0); yTo(0); });
  });
}
```

#### ПРИОРИТЕТ 6 — Custom Cursor

Только десктоп. Скрыть на тач-устройствах.

```javascript
// CSS: cursor: none на body (только desktop)
// JS: div.cursor следует за мышью через GSAP quickTo
const cursor = document.querySelector('.cursor');
const xTo = gsap.quickTo(cursor, 'x', { duration: 0.2, ease: 'power3.out' });
const yTo = gsap.quickTo(cursor, 'y', { duration: 0.2, ease: 'power3.out' });

window.addEventListener('mousemove', e => { xTo(e.clientX); yTo(e.clientY); });

// При наведении на изображения галереи — расширяется
document.querySelectorAll('.gallery__card img').forEach(img => {
  img.addEventListener('mouseenter', () => cursor.classList.add('cursor--expand'));
  img.addEventListener('mouseleave', () => cursor.classList.remove('cursor--expand'));
});
```

CSS:
```css
.cursor {
  position: fixed;
  width: 28px; height: 28px;
  border: 1.5px solid var(--color-accent);
  border-radius: 50%;
  pointer-events: none;
  z-index: 9999;
  transform: translate(-50%, -50%);
  transition: width 0.3s, height 0.3s;
```

## 9. ТРЕБОВАНИЯ К ПРОИЗВОДИТЕЛЬНОСТИ

| Метрика | Цель |
|---------|------|
| Lighthouse Performance | ≥ 90 |
| LCP (Largest Contentful Paint) | ≤ 2.5 сек |
| FID / INP | ≤ 100 мс |
| CLS | ≤ 0.1 |
| Загрузка на 4G | ≤ 3 сек |

**Обязательные оптимизации:**
- Все изображения — **WebP**, максимум 200KB для hero, 80KB для карточек
- Атрибут `loading="lazy"` на всех изображениях ниже первого экрана
- Hero-изображение — `loading="eager"` + `<link rel="preload">`
- Шрифты — `font-display: swap` + preconnect к fonts.googleapis.com
- GSAP и Lenis — динамический импорт (`import()`) после интерактивности страницы
- `will-change: transform` только на активно анимируемых элементах, удалять после завершения
- Все scroll listeners — `{ passive: true }`

```html
<!-- В <head> -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preload" as="image" href="/assets/images/hero.webp">
```

## 11. ЧЕГО НЕ ДЕЛАТЬ

### Дизайн
- ❌ Студийные фото кашпо на белом фоне
- ❌ Стоковые фото людей, растений, интерьеров
- ❌ Более одного акцентного цвета (только терракота)
- ❌ "Продающие" паттерны: таймеры обратного отсчёта, всплывающие скидки, мигающие баннеры
- ❌ Carousel/slider в герое

### Тексты
- ❌ "Мы стремимся к качеству" и любые аналогичные клише
- ❌ "Цена по запросу" без хотя бы диапазона
- ❌ "Купить сейчас" — слишком транзакционно для кастома
- ❌ Третье лицо ("мастер делает") вместо первого ("я делаю")

### Анимации
- ❌ `bounce` или `elastic` easing — воспринимается как дешёвое и пластиковое
- ❌ Scroll-jacking (полный контроль скролла, когда пользователь не управляет скоростью)
- ❌ Hover-эффекты на мобайле без `@media (hover: hover)` — дают "phantom hover"
- ❌ Preloader длиннее 1.5 сек — bounce rate растёт
- ❌ Одновременно 3+ анимации в одной секции — глаз не знает, куда смотреть

### UX
- ❌ Форма с более чем 3 полями как основной CTA
- ❌ Email как единственный контакт — российская аудитория пишет в мессенджеры
- ❌ Скрытые цены — снижает доверие у аудитории 25–35 лет
